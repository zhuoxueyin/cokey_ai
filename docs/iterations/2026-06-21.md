- **Skill 库代码库同步 Tag**
  - `drama_skill_service.compute_repo_sync_meta`：对比 `skills/` 与 Skill 库最新草稿
  - 列表 API 返回 `repo_has_changes` / `repo_version` / `repo_folder`
  - `SkillAdminPage`：「代码库」列 Tag + 顶部未同步汇总 Alert

## 修复 / 优化

- 系统模板明确：非 character 阶段禁止 JSON/YAML 外露
- 单元测试：`test_agent_creation_pipeline` / `test_concept_reply_normalize` / `test_drama_skill_service`（repo sync meta）
- **创作助手无响应 / 重复提问**
