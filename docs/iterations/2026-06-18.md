## 修复 / 优化

| 问题 | 处理 |
|------|------|
| 粘贴失败 HTTP 404 | 根因：运行中 backend 未加载 `duplicate` 路由 → `launcher restart`；前端增加 404 提示 |
| 视频下载失败 | 外部 URL 无 CORS → 后端流式代理 + 前端 fallback 新标签页 |
| 渠道 curl 弹窗被挡 | curl Modal 嵌套进编辑弹窗并提高 zIndex |
| 世界杯等项目不在列表 | `user_id` 过滤过严 → 列表放宽 + 打开时认领 |
