## 修复 / 优化

| 问题 | 处理 |
|------|------|
| 粘贴失败 HTTP 404 | 根因：运行中 backend 未加载 `duplicate` 路由 → `launcher restart`；前端增加 404 提示 |
| 视频下载失败 | 外部 URL 无 CORS → 后端流式代理 + 前端 fallback 新标签页 |
| 渠道 curl 弹窗被挡 | curl Modal 嵌套进编辑弹窗并提高 zIndex |
| 世界杯等项目不在列表 | `user_id` 过滤过严 → 列表放宽 + 打开时认领 |
| 任务管理页表头错位 | TaskAdmin Card extra 挤压表格 → 筛选区内嵌 + 去 fixed 列 |
| 运行记录无法选中复制 | Typography copyable → 可选中 `<code>` + 独立复制按钮 |
| 标记笔迹与鼠标错位 | 用 canvas `getBoundingClientRect` 归一化坐标 |
| 擦除保存后大片黑色 | 导出改为底图 + 独立标记层合成 |
| 画布文本效果差于创作流 | 对齐 `_build_run_params`：prompt/images 与 Workspace 一致 |
