# 一、技能简介
标准化生成上美影《天书奇谭》80年代国风手绘画面，统一泥塑卡通造型、粗黑轮廓、平涂年画配色，规避现代CG、写实、渐变光影，输出纯正中国学派老动画效果。

# 二、适用场景
1. 需要复古上美老动画、民间泥塑怪诞国风画面
2. 蛋生、狐妖、古寺山水国风插画制作
不适用场景：写实人像、3D科幻、二次元日系画风

# 三、输入条件
1. 输入主体关键词：人物/场景描述
2. 固定风格前置词：系统固定加载，无需用户重复输入

# 四、核心执行流程
1. 加载固定上美天书奇谭风格基底词
2. 拼接用户主体需求词
3. 追加画质与构图参数
4. 叠加负面规避词
5. 生成并校验画面风格合规性

# 五、输出标准
1. 画风：80年代赛璐璐手绘、泥塑造型、粗黑轮廓、纯色平涂
2. 无3D、无写实、无渐变、无高光
3. 比例16:9/1:1可选，胶片复古颗粒质感

# 六、异常兜底
1. 画面偏现代：强化负面词二次元、CG、渐变
2. 线条过细：加厚轮廓权重

# 七、约束禁忌
禁止出现现代特效、写实光影、立体渲染、日系画风
